本地存储：
    本地存储为字符串格式
    let data = [{
        title: 'sh',
        id: 0
    },{
        title: 'hua',
        id:12
    }
    ];
    先把对象形式转为JSON字符串形式
    JSON.stringgify(obj)
    localStrage.setItem('data1',JSON.stringgify(obj))
    取出时是要再从字符串格式转换为对象格式
    let data1 = JSON.parse(localStorage.getItem('data1'));
    这里的data1是数组对象，即可取出对应的内容啦
